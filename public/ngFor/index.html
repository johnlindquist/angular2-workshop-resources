06. Repeating Elements with ngFor

Repeating Elements with *ngFor

TL;DR

  • New syntax *ngFor="#person of people"

Gotchas

  • It's of, not in
  • Remember the #

Show Me the Code

src/home/home.component.ts

import {Component} from 'angular2/core';
import {Person} from './components/person.component';

@Component({
    selector: 'home',
    directives: [Person],
    template: `
        <ul>
            <li *ngFor="#person of people">
                {{person.name}}
            </li>
        </ul>

        <person 
            *ngFor="#person of people" 
            [name]="person.name">            
        </person>
    `
})
export class Home {
    people = [
        {name: "Sally"},
        {name: "James"},
        {name: "Percy"},
        {name: "Claire"},
        {name: "Izzie"}
    ];
}

src/home/components/person.component.ts

import {Component, Input} from 'angular2/core';
@Component({
    selector: 'person',
    template: `
        <style>
        .person{
            font-family: Helvetica,serif;
            border-top: 2px dashed cornflowerblue;
            background-color: lightyellow;
        }
        </style>

        <div class="person">
            {{name}}
        </div>
    `
})
export class Person{
    @Input() name;
}

What's That Syntax?

The * Star

The asterix/star/whatever you want to call it, represents a template element. In an *ngFor, the * means to "use this element as a template and repeat it".
