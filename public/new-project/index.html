00. New Project

Creating a New Project From Scratch

The Angular CLI

The angular-cli will help you get your Angular 2 project up and running quickly.

https://github.com/angular/angular-cli

You can follow the instructions there to get it installed.

The Angular Seed Project

If you prefer to use WebPack or have more control, there's also https://github.com/angular/angular2-seed.

Both approaches work great and can generate a build for you, so I'll leave it up to your preference on how you want to manage your project.
